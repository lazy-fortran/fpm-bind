# fpm-bind Examples

Each subdirectory is a standalone fpm project demonstrating fpm-bind usage.

## Running an example

```bash
cd simple
fpm bind python
ls build/bind/python/
```

## Examples

### simple/
Basic functions and subroutines with scalar arguments.

### arrays/
Array operations: dot product, scaling, matrix-vector multiplication.

### derived_types/
Derived types (point_t, circle_t) with initialization and methods.
